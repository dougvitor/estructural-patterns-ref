# Padrões Estruturais.

Esta categoria apresenta como é possível estruturar diversos objetos e classes de forma extensível e flexível.
Foram abordados os seguintes padrões:
- Adapter;
- Bridge;
- Composite;
- Decorator;
- Facade;
- Flyweight;
- Mediator;
- Proxy.
